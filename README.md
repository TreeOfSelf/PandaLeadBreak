# PandaLeadBreak

## Description

A simple drop-in mod to make leads strong enough to pull entities while flying around on the elytra.

## Try it out
`hardcoreanarchy.gay`   (Deathban Anarchy)  
`sky.hardcoreanarchy.gay`   (Skyblock Anarchy)
